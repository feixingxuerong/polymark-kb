# 航空天气信号定义

> 将航班运行数据量化为天气市场信号的方法论

---

## 概述

本文档定义可用于 Polymarket 天气预测市场的航空数据信号，说明：
- 可量化指标的计算方法
- 适用的天气市场类型
- 信号解释的边界条件
- 常见失败模式与排除方法

---

## 核心信号定义

### 1. 机场 Disruption Index

**计算方式**：
```
Disruption Index = f(延误航班数, 平均延误时长, 取消航班数)
范围: 0.0 - 5.0
```

**数据来源**：Flightradar24 Airport Disruption Map

**适用天气市场**：
- ⭐ 大雾/低能见度
- ⭐⭐ 雷暴
- ⭐⭐⭐ 飓风/台风登陆
- ⭐⭐⭐ 暴雪

**信号解读**：
- 0.0-1.9: 天气影响轻微
- 2.0-3.4: 天气导致部分延误/取消
- 3.5-5.0: 严重天气中断，大规模取消

---

### 2. 航班取消率 (Cancellation Rate)

**计算方式**：
```
取消率 = 当日取消航班数 / 当日计划航班数 × 100%
```

**数据来源**：
- Flightradar24 Stats
- Aviationstack API
- FAA OPSNET

**适用天气市场**：
- ⭐⭐⭐ 大雾/低能见度
- ⭐⭐⭐ 雷暴（对流天气）
- ⭐⭐⭐⭐ 飓风/台风
- ⭐⭐⭐⭐ 暴风雪

**信号强度**：
- < 1%: 正常波动
- 1-5%: 中等影响
- > 5%: 重大天气事件

---

### 3. 航班延误分位数 (Delay Percentiles)

**计算方式**：
```
- P50 (中位数延误): 50%航班延误时长以下
- P90 (90分位): 仅10%航班延误超过此值
- 平均延误: 总延误分钟 / 延误航班数
```

**数据来源**：FAA OPSNET、BTS

**适用天气市场**：
- ⭐⭐ 雷暴（对流延误）
- ⭐⭐⭐ 暴雪（地面除雪导致延误）
- ⭐⭐⭐ 大雾（能见度低于ILS落地标准）

**信号解读**：
- P90 > 60分钟: 显著天气影响
- P90 > 120分钟: 严重天气事件

---

### 4. 地面停航率 (Ground Stop Rate)

**计算方式**：
```
地面停航 = 受地面停止(Ground Stop)影响的航班数
或: 机场关闭时长
```

**数据来源**：FAA OPSNET、机场NOTAMs

**适用天气市场**：
- ⭐⭐⭐⭐ 飓风登陆（大规模地面停航）
- ⭐⭐⭐⭐ 暴风雪（跑道关闭）
- ⭐⭐⭐ 大雾（能见度低于最低标准）

**信号强度**：地面停航是天气影响的强信号，通常意味着机场实际关闭或严重限制

---

### 5. 航路绕飞距离 (Detour Distance)

**计算方式**：
```
绕飞距离 = 实际飞行距离 - 大圆距离(直飞距离)
正值表示绕飞，数值越大表示天气影响越严重
```

**数据来源**：
- FlightAware（航路历史）
- ADS-B Exchange（历史轨迹）
- OpenSky Network

**适用天气市场**：
- ⭐⭐⭐ 雷暴（绕飞对流云团）
- ⭐⭐ 飓风外围云系
- ⭐⭐ 火山灰/沙尘暴

**信号解读**：
- 绕飞距离 < 50km: 轻微天气影响
- 绕飞距离 50-200km: 中等绕飞
- 绕飞距离 > 200km: 严重天气系统

---

### 6. 进近失败率 (Missed Approach Rate)

**计算方式**：
```
进近失败率 = 进近失败次数 / 总进近次数 × 100%
```

**数据来源**：FAA OPSNET、机场ATIS

**适用天气市场**：
- ⭐⭐⭐ 大雾/低能见度
- ⭐⭐⭐ 侧风/阵风
- ⭐⭐⭐ 暴风雨

**信号强度**：进近失败直接反映天气低于落地标准

---

### 7. 航班流量变化率 (Traffic Flow Change)

**计算方式**：
```
流量变化率 = (当日实际航班数 - 历史同期平均) / 历史同期平均 × 100%
```

**数据来源**：
- Flightradar24 Stats
- EUROCONTROL
- FAA OPSNET

**适用天气市场**：
- ⭐⭐⭐⭐ 飓风预警（提前1-3天流量下降）
- ⭐⭐⭐⭐ 暴风雪预警
- ⭐⭐ 重大雷暴

**信号解读**：负向大幅变化通常表示天气事件的预期影响

## 失败模式与排除方法

### 1. 航空管制 (Air Traffic Control)

**问题**：空域管制、空中交通拥堵、军事活动限制导致的延误/取消，非天气原因

**排除方法**：
- 查看 FAA OPSNET 延误原因分类（Weather vs Carrier Delay vs NAS Delay）
- 对比同一区域多个机场表现
- 检查是否有NOTAMs军事活动公告

---

### 2. 节假日/大型活动

**问题**：感恩节、圣诞节、新年等假期出行高峰导致的延误增加，非天气

**排除方法**：
- 对比历史同期（相同星期几、相同假期）
- 查看BTS历史节假日流量数据
- 使用流量变化率时需校正节假日效应

---

### 3. 航空公司运营原因

**问题**：机组超时、机械故障、罢工等导致的取消和延误

**排除方法**：
- FAA OPSNET 提供 Carrier Delay 分类
- 对比多家航空公司表现（某一家故障 vs 普遍天气）
- 查看是否有机票超售、机组问题新闻

---

### 4. 机场设施问题

**问题**：跑道维护、航站楼问题、停电等导致的限制

**排除方法**：
- 检查机场NOTAMs维护通告
- 查看ADIP机场数据确认设施状态
- 对比附近替代机场表现

---

### 5. 区域 vs 局部影响

**问题**：信号反映的是区域天气还是单一机场问题

**排除方法**：
- 多机场交叉验证
- 关注目标市场对应的具体机场
- 结合地面观测/雷达数据

## 信号融合方法

### 组合信号权重建议

**飓风/台风市场**：
1. 航班流量变化率 (30%) - 预警期领先指标
2. 取消率 (30%) - 事件期核心指标
3. 地面停航 (25%) - 登陆期强信号
4. Disruption Index (15%) - 快速参考

**暴雪市场**：
1. 取消率 (35%) - 核心
2. 地面停航/跑道关闭 (30%)
3. 延误分位数P90 (20%)
4. Disruption Index (15%)

**雷暴市场**：
1. 绕飞距离 (30%) - 对流云特征
2. 延误分位数P90 (25%)
3. 取消率 (20%)
4. Disruption Index (25%)

**大雾市场**：
1. 进近失败率 (35%) - 直接反映能见度
2. Disruption Index (30%)
3. 取消率 (20%)
4. 延误分位数 (15%)

## 快速检查清单

使用航班数据辅助天气市场决策时的检查流程：

- [ ] **确认天气事件存在**：查看地面观测、卫星、雷达
- [ ] **排除非天气因素**：检查NOTAMs、节假日、管制
- [ ] **多信号交叉验证**：至少2个独立信号一致
- [ ] **时间窗口匹配**：航班数据时间与市场结算时间匹配
- [ ] **区域特异性**：关注目标市场地理区域的机场数据
- [ ] **对比历史**：当前信号 vs 同类天气历史表现
