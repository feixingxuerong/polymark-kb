# 航空数据源清单

> 航班/航空运行数据作为 Polymarket 天气市场辅助信号的可用数据源汇总

---

## 概述

本文档整理了可用于天气市场 nowcasting 的航空数据源，涵盖：
- 实时航班追踪
- 历史航班统计
- 机场延误/取消数据
- 商业航班 API

**评估维度**：覆盖范围、更新频率、回溯能力、限制/合规、适用场景

---

## 数据源列表

### 1. FAA OPSNET / Air Traffic Activity Data System

| 项目 | 内容 |
|------|------|
| **URL** | https://aspm.faa.gov/opsnet/sys/Main.asp |
| **类型** | 政府公开数据 |
| **覆盖范围** | 美国国内机场、航班起降、延误统计 |
| **更新频率** | 每日/每月 |
| **可回溯** | 是，历史数据可追溯数年 |
| **限制/合规** | 公开数据，无商业使用限制 |
| **适用场景** | 美国机场天气影响评估、延误分析 |

**备注**：提供机场起降架次、延误原因分类（天气、机械、管制等），适合判断天气导致的航班中断。

---

### 2. FAA Airport Data (ADIP)

| 项目 | 内容 |
|------|------|
| **URL** | https://adip.faa.gov/agis/public/ |
| **类型** | 政府公开数据 |
| **覆盖范围** | 美国所有注册机场元数据、设施信息 |
| **更新频率** | 不频繁 |
| **可回溯** | 是 |
| **限制/合规** | 公开数据，无限制 |
| **适用场景** | 机场基础信息查询、跑道/设施信息 |

---

### 3. Flightradar24 航班统计

| 项目 | 内容 |
|------|------|
| **URL** | https://www.flightradar24.com/data/statistics |
| **类型** | 商业平台（免费+付费） |
| **覆盖范围** | 全球商业航班、每日/每月飞行统计 |
| **更新频率** | 每日更新 |
| **可回溯** | 部分回溯（约4年历史） |
| **限制/合规** | 需遵守使用条款，商业使用需授权 |
| **适用场景** | 全球航班量趋势、疫情影响分析 |

**备注**：提供商业航班总数、民航航班数对比，适合宏观趋势判断。

---

### 4. Flightradar24 Airport Disruption Map

| 项目 | 内容 |
|------|------|
| **URL** | https://www.flightradar24.com/data/airport-disruption |
| **类型** | 商业平台（免费+付费） |
| **覆盖范围** | 全球约300个繁忙机场的延误/取消指数 |
| **更新频率** | 实时 |
| **可回溯** | 可查看历史某一天数据 |
| **限制/合规** | 需遵守使用条款 |
| **适用场景** | 实时机场 disruption 监控、天气影响快速评估 |

**核心指标**：Disruption Index (0-5.0)
- 0.0-1.9: 良好
- 2.0-3.4: 轻微问题
- 3.5-5.0: 重大中断

---

### 5. FlightAware ADS-B

| 项目 | 内容 |
|------|------|
| **URL** | https://www.flightaware.com/adsb/ |
| **类型** | 商业平台（免费+付费） |
| **覆盖范围** | 全球ADS-B/Mode S航班追踪、70%全球客机 |
| **更新频率** | 实时（每秒更新） |
| **可回溯** | 需付费订阅历史数据 |
| **限制/合规** | 商业使用需授权 |
| **适用场景** | 实时航班追踪、航路分析、绕飞检测 |

**备注**：拥有超过45个数据源，包括政府ATC数据，全球最大免费航班追踪平台。

---

### 6. ADS-B Exchange

| 项目 | 内容 |
|------|------|
| **URL** | https://www.adsbexchange.com/data/ |
| **类型** | 商业数据服务 |
| **覆盖范围** | 全球、包含军航、政府飞机等非商业航班 |
| **更新频率** | 实时（最高2Hz） |
| **可回溯** | 是，历史数据文件可下载 |
| **限制/合规** | 商业使用需书面授权，禁止批量转售 |
| **适用场景** | 全类型航班追踪、军航/政府飞机数据 |

**独特价值**：提供其他商业平台过滤的军航、政府VIP、禁飞列表飞机数据。

---

### 7. Aviationstack API

| 项目 | 内容 |
|------|------|
| **URL** | https://aviationstack.com/ |
| **类型** | 商业API（免费Tier可用） |
| **覆盖范围** | 全球13,000+航空公司、10,000+机场 |
| **更新频率** | 实时（API调用） |
| **可回溯** | 是，历史航班数据 |
| **限制/合规** | 付费订阅适用商业条款 |
| **适用场景** | 程序化获取航班状态、机场时刻表、飞机注册信息 |

**定价**：免费版100次/月，付费版$49.99/月起

## 快速接入建议

### 优先级 1：实时 disruption 监控
- **Flightradar24 Airport Disruption Map** - 最适合天气事件快速评估

### 优先级 2：程序化数据获取
- **Aviationstack API** - 免费 tier 可用于测试，适合程序化集成

### 优先级 3：美国航班延误分析
- **FAA OPSNET** + **BTS** - 官方数据，权威性强

### 优先级 4：全球航班追踪（研究）
- **OpenSky Network** - 学术用途免费
