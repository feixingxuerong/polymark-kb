# 风险框架 + 资金管理方案

> 本文档定义 Polymarket 交易的风险分层体系与资金管理规则

---

## 1. 风险分层模型

### 1.1 五层风险架构

| 层级 | 风险类型 | 描述 | 控制手段 |
|------|----------|------|----------|
| **L1 账户风险** | 整体账户安全 | API 密钥泄露、账户被禁 | 最小权限 API Key、IP 白名单 |
| **L2 执行风险** | 交易执行层面 | 滑点、延迟、部分成交 | 限价单、冰山订单 |
| **L3 策略风险** | 策略逻辑缺陷 | 模型错误、过拟合 | 回测验证、单策略最大敞口 |
| **L4 市场风险** | 市场固有风险 | 流动性枯竭、极端波动 | 最小流动性阈值、持仓分散 |
| **L5 合规风险** | 法律与监管 | 地区限制、税务 | KYC 检查、地区过滤 |

## 2. 仓位与资金管理

### 2.1 Kelly Criterion

**标准 Kelly Formula:**
```
f* = (bp - q) / b
```
- `b` = 净赔率 (decimal odds - 1)
- `p` = 胜率估算
- `q` = 1 - p

**保守版本 (Half-Kelly):**
- 使用 `f* = 0.5 × 标准 Kelly` 以降低方差
- 适合追求稳定增长 vs 最大化收益

### 2.2 固定比例模型

| 模型 | 公式 | 适用场景 |
|------|------|----------|
| **固定分数 (Fixed Fraction)** | 每笔投入 = 账户 × 固定% | 初始阶段、低波动市场 |
| **固定比例 (Fixed Ratio)** | 每笔投入 = 账户 / N | 需动态调整 Kelly |
| **波动率调整** | 根据最近 N 天波动率缩放仓位 | 高波动期降低仓位 |

### 2.3 止损与回撤控制

| 规则 | 阈值 | 动作 |
|------|------|------|
| **单笔止损** | -100% 头寸价值 | 自动平仓 |
| **单日回撤** | -5% 账户日跌幅 | 停止新开仓 |
| **单周回撤** | -10% 账户周跌幅 | 全面停手，复盘 |
| **最大回撤** | -20% 账户历史最高 | 强制清仓，重新评估 |

## 3. 授权边界

### 3.1 必需用户确认的场景

以下操作**必须**获得用户 explicit 确认：

- [ ] 首次授权资金账户
- [ ] 单日亏损超过 3% 后的新开仓
- [ ] 任何策略参数变更
- [ ] 超过单笔最大风险阈值的仓位
- [ ] 达到最大回撤后的重启交易

### 3.2 可自动执行

- 止损单挂单
- 流动性检查
- 赔率监控
- 符合预设参数的新开仓

## 4. 监控指标

### 4.1 核心监控仪表盘

| 指标 | 计算方式 | 告警阈值 |
|------|----------|----------|
| **当前风险暴露** | sum(每笔仓位 × 赔率) / 账户 | > 50% |
| **日盈亏** | 当日平仓盈亏 + 未平浮动 | ±3% |
| **Sharpe Ratio (滚动)** | (日均收益 / 日波动) × √252 | < 1.0 |
| **最大连续亏损** | 历史上连续亏损次数 | > 5 次 |
| **持仓分散度** | HHI (持仓集中度指数) | > 0.5 |

### 4.2 健康状态检查

- ✅ 正常运行：所有指标在阈值内
- ⚠️ 预警：任一指标突破告警阈值
- 🔴 停止：达到强制止损或最大回撤

## 5. 默认参数建议 (Medium Risk)

以下为 **medium risk** 风险偏好的默认参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **单笔最大风险** | 2% 账户 | 单笔交易最大亏损占账户比例 |
| **单日最大回撤** | 5% | 日终账户跌幅超 5% 停止交易 |
| **单周最大回撤** | 10% | 周终账户跌幅超 10% 全面停手 |
| **最大持仓数** | 10 个市场 | 同时持仓的最大市场数量 |
| **最小流动性阈值** | $1000 | 市场最小交易量要求 |
| **允许滑点阈值** | 2% | 超过则拒绝成交 |
| **Kelly 系数** | 0.25 (Half-Kelly) | 使用 25% Kelly fraction |
| **止损线** | -100% 单笔 | 单笔全损自动平仓 |

## 6. 用户决策点

以下参数需要用户确认后生效：

1. **总预算/账户上限** - 投入 Polymarket 的总资金量
2. **最大回撤容忍度** - 上述 20% 是否可接受，或需更保守
3. **持仓周期偏好** - 短线 (日内) / 中线 (1-7天) / 长线 (周级别)
4. **策略数量上限** - 同时运行多少个独立策略
5. **杠杆使用** - 是否允许杠杆，若允许上限多少
